# 🛰️ Yakami Eco-Gateway: O Hub da Economia Circular Fluvial

O **Eco-Gateway** é a solução da YAKAMI TECH para transformar conectividade via satélite (Starlink) em uma plataforma de monetização e fidelidade para a Bacia Amazônica.

## 🚀 Proposta de Valor
Nossa solução resolve o alto custo de conectividade para donos de embarcações e a falta de serviços digitais para passageiros através de:
- **Hotspot Web3:** Acesso Wi-Fi automatizado via pagamento por pontos.
- **YAKAMI Points (YKM):** Programa de fidelidade tokenizado para troca de serviços entre barcos, lanchas e pousadas.
- **Marketplace Offline:** Conteúdo local servido diretamente do gateway, otimizando o uso da banda de satélite.

## 🛠️ Arquitetura do Projeto
- **Firmware:** OpenWrt + Captive Portal customizado.
- **Blockchain:** Smart Contracts em Solidity (Rede Sepolia/Polygon) para gestão de pontos.
- **Frontend:** Portal de usuário responsivo (Mobile First) para ambientes de baixa latência.
- **Integração:** API de pagamentos para conversão instantânea PIX -> Y-Points.
